# 方法 · 马斯克假设爆破四步

## 四步动作

1. 识别神圣假设：列出该行业所有人默认但不质疑的前提。
2. 第一性拆到原子：不问市场怎么定价，问原子/元素周期表怎么定价，找成本黑洞（成品溢价数十倍处）。
3. 重定义问题：把成本黑洞 + 拔掉的假设，翻译成一个被重新定义的问题。
4. 终局倒推：推演 3-5 年后假设塌掉后的现实（愿景），再倒推今天瞄准哪里。

## 输出：颠覆式候选七要素（缺一不立档，存为 候选档案\CAND-NNN-*.md）

1. 拔掉的神圣假设
2. 第一性拆解（原子成本 / 成本黑洞）
3. 重定义的问题
4. 3-5 年终局愿景
5. 同位墓碑（谁试过 / 为何死 / 窗口变了什么）
6. 类型判定（S032 类型 A 低端破坏 / 类型 B 基础设施重构）+ 证据等级 A/B/C/X
7. 破坏式滩头第一步 → 交 SK（只给指针，不展开）
